  <behavior>
    - resolveJsonPath("{"temperature": 22.5}", "temperature") returns "22.5"
    - resolveJsonPath("{"data": {"value": 42}}", "data.value") returns "42"
    - resolveJsonPath("not json", "foo") returns "not json" (fallback)
    - resolveJsonPath("{"a": 1}", "missing") returns "{"a": 1}" (fallback)
    - resolveJsonPath("{"a": null}", "a") returns "{"a": null}" (null fallback)
    - applyDisplayTemplate("{{value}} °C", "22.5") returns "22.5 °C"
    - applyDisplayTemplate("", "22.5") returns "22.5" (empty template passthrough)
  </behavior>
  <action>
    1. Edit `src/types/settings.ts` — extend MqttActionSettingsSchema with these NEW fields (keep all existing fields unchanged):
       - `brokerUsername: z.string().optional()` — MQTT auth username (UI-01)
       - `brokerPassword: z.string().optional()` — MQTT auth password (UI-01)
       - `brokerTls: z.boolean().default(false)` — TLS toggle (UI-01)
       - `jsonPath: z.string().optional()` — dot-notation path for JSON extraction (SUB-03, UI-03)
       - `displayTemplate: z.string().optional()` — template with {{value}} placeholder (D-17, UI-03)
       - `onPayload: z.string().optional()` — payload to publish for "turn on" (TOGL-01, UI-04)
       - `offPayload: z.string().optional()` — payload to publish for "turn off" (TOGL-01, UI-04)
       - `onValue: z.string().optional()` — subscribe value meaning "on" (TOGL-01, UI-04)
       - `offValue: z.string().optional()` — subscribe value meaning "off" (TOGL-01, UI-04)

    2. Create `src/util/resolve-json-path.ts` with two exported functions:
       - `resolveJsonPath(payload: string, path: string): string` — Parse payload as JSON, walk dot-separated path keys, return String(result). On any error (parse failure, missing key, null value), return the raw payload unchanged. No external dependencies. Per D-16, simple dot-notation only (no array indexing, no wildcards).
       - `applyDisplayTemplate(template: string, value: string): string` — If template is non-empty, return `template.replace("{{value}}", value)`. Otherwise return value unchanged. Per D-17, single {{value}} placeholder only.

    3. Create `src/util/resolve-json-path.test.ts` with vitest tests covering the behavior cases above.
  </action>
