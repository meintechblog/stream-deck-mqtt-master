       - Publish Topic: cmnd/testdevice/POWER

    5. From terminal: `mosquitto_pub -h 192.168.3.8 -t stat/testdevice/POWER -m "OFF"`
    6. Button should show "OFF" title and switch to state 0 (off icon)
    7. Press button again — should publish "ON"
